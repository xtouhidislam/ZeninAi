Ai Automation Agency

## Environment variables
Copy .env.example to .env.local and fill in the EmailJS values before running locally.

For Vercel deployment, add these environment variables in the project settings:
- NEXT_PUBLIC_EMAILJS_SERVICE_ID
- NEXT_PUBLIC_EMAILJS_TEMPLATE_ID
- NEXT_PUBLIC_EMAILJS_PUBLIC_KEY
- NEXT_PUBLIC_CONTACT_EMAIL
